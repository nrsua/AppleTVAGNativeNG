# AppleTVAGNativeNG

## Что делает плагин

- добавляет собственный `topnav` поверх стандартной шапки Lampa
- показывает часы в правом верхнем углу
- позволяет включать и выключать плагин из настроек
- позволяет выбрать, какие пункты из `menu_list` попадут в верхнее меню
- добавляет эффект `Наклон veoveo.ru` от arabian_q
- перестраивает карточки каталога в горизонтальный формат, добавляет информацию на сами карточки с добавлением логотипов
- добавляет бейдж типа контента: `ФИЛЬМ` / `СЕРИАЛ`
- старается сохранить управление с пульта и клавиатуры без потери фокуса


## Совместимость

Плагин рассчитан в первую очередь на:

- браузерную версию Lampa
- устройства AppleTV
